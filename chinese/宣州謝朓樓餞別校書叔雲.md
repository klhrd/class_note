## 一、原文

```
棄我去者，昨日之日不可留；亂我心者，今日之日多煩憂。
長風萬里送秋雁，對此可以酣高樓。
蓬萊文章建安骨，中間小謝又清發。
俱懷逸興壯思飛，欲上青天覽明月。
抽刀斷水水更流，舉杯消愁愁更愁。
人生在世不稱意，明朝散髮弄扁舟。
```


## 二、白話翻譯（逐段）

```
離我而去的昨天早已無法挽留，擾亂我心情的今天卻充滿煩惱與憂愁。
長風萬里送走秋天的大雁，面對這樣的景象，正適合在高樓暢快飲酒。
你的文章如同仙境般高妙，又具有建安時代雄健的風骨，同時還承襲謝朓清新俊逸的文風。
我們都懷著超脫的興致與壯闊的理想，恨不得飛上青天，一覽明月。
可是拔刀斬水，水反而流得更急；舉杯想消愁，愁卻更加深重。
既然人生在世多半不如意，不如明天披散頭髮，乘著小船遠走江湖。
```


## 三、賞析重點

### 1️⃣ 情感主軸：**理想受挫的豪放之愁**

這首詩表面是在「送別朋友」，實際上卻深藏李白**對現實不滿、對人生苦悶的抒發**。
他壯志未酬，仕途受阻，內心充滿矛盾與不甘。


### 2️⃣ 名句解析

#### 🔹「棄我去者，昨日之日不可留」

* 表達**對過去的無奈**
* 昨天已經過去，怎麼留也留不住

#### 🔹「抽刀斷水水更流，舉杯消愁愁更愁」

* 全詩最有名的句子
* 用誇張比喻說明：
  👉 **情緒無法靠外力（酒、逃避）解決**


### 3️⃣ 人物與文學典故

* **小謝**：指南朝詩人「謝朓」，以清新自然聞名
* 李白稱讚朋友文章「既有骨氣、又有才情」
* 表現出**文人彼此欣賞、惺惺相惜**


### 4️⃣ 結尾精神：李白式的灑脫

> 「明朝散髮弄扁舟」

* 不是逃避，而是**主動選擇自由**
* 即使現實不如意，也不向命運低頭
* 是李白典型的：**浪漫 × 反叛 × 自由精神**


## 四、整體評價（一句話）

> **這是一首表面送別、實則抒寫人生困境的豪放詩，
> 悲中有氣，愁中有夢，充滿李白獨有的浪漫與不屈。**

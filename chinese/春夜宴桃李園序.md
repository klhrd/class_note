## 一、原文

```
夫天地者，萬物之逆旅也；光陰者，百代之過客也。
而浮生若夢，為歡幾何？
古人秉燭夜遊，良有以也。

況陽春召我以煙景，大塊假我以文章。
會桃李之芳園，序天倫之樂事。
群季俊秀，皆為惠連；吾人詠歌，獨慚康樂。

幽賞未已，高談轉清。
開瓊筵以坐花，飛羽觴而醉月。
不有佳作，何伸雅懷？
如詩不成，罰依金谷酒數。
```


## 二、白話翻譯

```
天地，是萬物暫時停留的旅舍；時間，是千百世代匆匆而過的過客。
而短暫的人生就像一場夢，真正的歡樂又能有多少呢？
古人拿著燭火在夜裡遊玩，確實是有道理的。

更何況溫暖的春天以朦朧美景迎接我們，大自然也借給我們文采與靈感。
在桃李盛開的芳香園中聚會，鋪敘親族之間的天倫之樂。
眾位弟弟才華出眾，都可比作謝惠連；我吟詩作歌，卻獨自慚愧不如謝靈運。

幽靜的賞景興致尚未結束，清雅的談論愈發高妙。
於是擺設華美的宴席，坐在花間；酒杯飛傳，對月而醉。
如果沒有好詩，又怎能抒發高雅的情懷？
若是作詩不成，就依照金谷園的規矩，罰酒幾杯。
```


## 三、賞析重點

### 1️⃣ 情感主軸：**把握當下的人生哲學**

本文並非單純記錄宴會，而是藉夜宴抒發李白對**人生短暫、及時行樂**的深刻體悟。
在感嘆生命如夢的同時，他並不消極，而是積極擁抱當下的歡樂與創作。


### 2️⃣ 名句解析

#### 🔹「夫天地者，萬物之逆旅也；光陰者，百代之過客也」

* 把天地比喻成旅館，時間比喻成過客
* 說明**人生短暫、萬物皆過客**

#### 🔹「浮生若夢，為歡幾何？」

* 感嘆人生如夢
* 引出下文「及時行樂」的合理性


### 3️⃣ 人物與文學典故

* **惠連、康樂**：分別指南朝詩人謝惠連、謝靈運

* 李白稱讚晚輩才華出眾，卻自謙不如

* 展現其**豪放中帶自省的文人氣度**

* **金谷酒數**：典出石崇金谷園宴飲規矩

* 作詩不成即罰酒，增添宴會的文雅趣味


### 4️⃣ 文章風格與精神

* 駢散交錯，語言華美而不浮誇
* 融合**哲思、親情、自然、文學**
* 展現李白「樂中有悟、狂中有理」的特色

# **🌍 國中地理：植物與氣候帶分類總整理**

這份表格將幫助你根據氣候特徵快速聯想對應的植物及其產業意義。

### **1\. 熱帶氣候區 (低緯度，全年高溫)**

此區植物多為「熱帶栽培業」的主角，具備高產值與外銷導向。

| 氣候類型 | 代表植物 | 作物類型 | 地理筆記 |
| :---- | :---- | :---- | :---- |
| **熱帶雨林** | **油棕** | 經濟作物 (油料) | 提煉棕櫚油，印尼、馬來西亞為核心。 |
| **熱帶雨林** | **天然橡膠** | 經濟作物 (工業) | 輪胎原料，需高溫多雨環境。 |
| **熱帶雨林** | **可可** | 經濟作物 (飲料) | 巧克力原料，主要在西非與中南美。 |
| **熱帶季風** | **稻米** | **糧食作物** | 亞洲人口稠密區主食，需充足水熱。 |
| **熱帶季風** | **黃麻** | 經濟作物 (特用) | 恆河三角洲特產，編織袋原料。 |
| **熱帶莽原** | **甘蔗** | 經濟作物 (糖料) | 喜乾濕分明，巴西、古巴、台灣中南部。 |
| **熱帶莽原** | **咖啡** | 經濟作物 (飲料) | 巴西高原與東非高原為重要產地。 |

### **2\. 溫帶氣候區 (中緯度，四季分明)**

此區植物與人類文明、工業發展及飲食習慣密切相關。

| 氣候類型 | 代表植物 | 作物類型 | 地理筆記 |
| :---- | :---- | :---- | :---- |
| **溫帶地中海型** | **橄欖** | 經濟作物 (油料) | 夏乾冬雨，具蠟質硬葉以抗旱。 |
| **溫帶地中海型** | **葡萄** | 經濟作物 (蔬果) | 釀酒工業基礎，喜夏季陽光充足。 |
| **溫帶季風/大陸** | **小麥** | **糧食作物** | 耐寒耐旱，「南稻北麥」的北方代表。 |
| **溫帶季風/大陸** | **甜菜** | 經濟作物 (糖料) | **「南蔗北菜」**，北方高緯製糖來源。 |
| **溫帶季風/大陸** | **棉花** | 經濟作物 (纖維) | 採收期忌雨，中國新疆與美國南方。 |
| **溫帶海洋型** | **牧草** | 經濟作物 (飼料) | 終年溫和多雨，發展乳畜業。 |
| **溫帶/副熱帶** | **茶** | 經濟作物 (飲料) | 喜排水良好的丘陵坡地。 |

### **3\. 乾燥與高山氣候區 (極端環境)**

在惡劣環境下生長的指標性植物，通常是當地居民的生存依賴。

| 氣候類型 | 代表植物 | 作物類型 | 地理筆記 |
| :---- | :---- | :---- | :---- |
| **乾燥氣候 (綠洲)** | **椰棗** | **糧食作物** | 耐旱耐鹽，西亞與北非居民的主食。 |
| **乾燥氣候 (沙漠)** | **仙人掌** | 自然植被 | 針狀葉減少蒸散，具儲水能力。 |
| **高山氣候** | **青稞** | **糧食作物** | 青藏高原特產，耐高寒貧瘠。 |

### **4\. 寒帶氣候與特定生態區**

| 氣候類型 | 代表植物 | 植物類別 | 地理筆記 |
| :---- | :---- | :---- | :---- |
| **寒帶氣候 (泰加林)** | **針葉林** | 林業資源 | 軟木來源，分布於俄羅斯、加拿大。 |
| **海岸潮間帶** | **紅樹林** | 生態植被 | 具支持根，防風定砂、保護海岸生態。 |

### **💡 考試必看：作物對照與關鍵詞**

* **糖料對照：** 甘蔗 (熱帶/南部) vs 甜菜 (溫帶/北部)。  
* **糧食對照：** 水稻 (濕熱/平原) vs 小麥 (乾冷/高原/平原)。  
* **纖維對照：** 棉花 (需晴朗採收) vs 黃麻 (需濕熱低窪)。  
* **飲料對照：** 茶 (丘陵) vs 咖啡 (熱帶高原) vs 可可 (雨林)。
